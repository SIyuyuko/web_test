# app // 项目名称

## Project setup // 项目配置安装

```bash
npm install
```

## Compiles and hot-reloads for development // 项目开发

```bash
npm run dev
```

## Compiles and minifies for production // 项目打包

```bash
npm run build
```

## Lints and fixes files // 修复项目文件

```bash
npm run lint
```
